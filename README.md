# Ritmo

Aplicativo pessoal para organizar hábitos, tarefas, treinos e metas.

## Como abrir

O projeto é um aplicativo web instalável (PWA). Sirva esta pasta por um servidor local e abra o endereço exibido no navegador do computador ou celular.

Os dados são armazenados no próprio navegador, sem necessidade de cadastro.

## Funcionalidades atuais

- Resumo do dia
- Hábitos com marcação de conclusão
- Tarefas com prioridade, prazo e lembretes
- Metas com prazo e progresso
- Exercícios editáveis
- Séries com peso, repetições e conclusão individual
- Adição e remoção de séries e exercícios
- Tema claro e escuro
- Instalação como aplicativo no celular
- Salvamento automático no aparelho
- Backup e restauração em arquivo
- Lembretes para hábitos, tarefas e treinos

## Notificações

Os lembretes locais são verificados enquanto o aplicativo está aberto ou ativo. Os navegadores exigem HTTPS para permitir notificações em celulares. Ao abrir por um endereço local `http://`, os dados continuam sendo salvos normalmente, mas as notificações podem ficar bloqueadas pelo navegador.
